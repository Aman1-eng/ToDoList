 To-Do List App

 Overview

The **To-Do List App** is a simple Java console-based application that helps users organize and manage their daily tasks. It allows users to add new tasks, view all existing tasks, update their completion status, and delete tasks through an easy-to-use menu-driven interface.

This project is designed for beginners learning Java programming concepts such as object-oriented programming, methods, loops, conditional statements, and the `ArrayList` collection.

--- Objectives

* Develop a console-based task management application.
* Practice Java programming fundamentals.
* Implement CRUD (Create, Read, Update, Delete) operations.
* Store and manage multiple tasks using `ArrayList`.
* Display task information in a clear and structured format.

---
 Features

* Add a new task with a short description.
* View all tasks with:

  * Task Number
  * Task Description
  * Task Status (Pending/Completed)
* Mark tasks as **Completed** or **Pending**.
* Delete tasks from the list.
* User-friendly menu-driven interface.
* Handles multiple tasks efficiently using `ArrayList`.

## How to Run

1. Open **Eclipse IDE**.
2. Create a new Java Project named **ToDoListApp**.
3. Add the Java source files to the **src** folder.
4. Save the project.
5. Right-click **ToDoListApp.java**.
6. Select **Run As тЖТ Java Application**.
7. Use the menu options to manage your tasks.
